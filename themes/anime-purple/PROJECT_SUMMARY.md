## 🚀 使用流程

### 安装
1. 克隆或下载主题到 `themes/anime-purple`
2. 在 Hexo `_config.yml` 中设置 `theme: anime-purple`
3. 编辑 `themes/anime-purple/_config.yml`
4. 运行 `hexo clean && hexo generate && hexo server`

### 自定义
1. 修改 CSS 变量改变色彩
2. 上传动漫壁纸到 `source/images/`
3. 配置社交链接
4. 可选：安装搜索、评论插件

### 部署
1. 运行 `hexo deploy`
2. 或部署到任何静态托管服务
